## Record & Clean (Real-time)

# Record 5s, suppress misc, save to processed
python -m ai.ai_runtime.audio.recorder_cleaner --duration 5 --suppress misc --threshold 0.03 --output ai/data/audio/processed/session_clean_misc.wav

# Record 10s, suppress pets, use device 2 (e.g. Virtual Cable)
python -m ai.ai_runtime.audio.recorder_cleaner --duration 10 --suppress pets --device 2

# List audio devices
python ai/scripts/demos/demo_custom_realtime.py --list-devices

## Batch Process (Offline)

python -m ai.ai_runtime.batch.batch_processor --input ai/data/audio/raw/keyboard_barking.wav --output ai/data/audio/processed/keyboard_barking_clean.wav --suppress pets,typing --threshold 0.03

## Virtual Cable Test Workflow

Use VB-Audio Virtual Cable to loop a WAV file into the suppression pipeline.

1. Find device ID:
   python ai/scripts/demos/demo_custom_realtime.py --list-devices
   (Look for "CABLE Output" – e.g. ID 2)

2. Terminal A – start loopback:
   python ai/scripts/demos/virtual_mic_streamer.py --input ai/data/audio/raw/speech_barking.wav

3. Terminal B – capture from Virtual Cable:
   python -m ai.ai_runtime.audio.recorder_cleaner --duration 10 --suppress pets --device 2

Without --device, scripts use the system default mic. If "CABLE Output" is not the default, they will record silence from your physical mic instead of the virtual stream.
